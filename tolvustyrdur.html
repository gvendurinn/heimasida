Dagbók

7.apríl

Allir pantaðir íhlutir sóttir. Einnig var byrjað að setja upp skjal með CAN-bus kóðum sem var hægt að finna á netinu. Vélartölvan og rafmangsstýrisdælan nota mismunandi CAN tungumál en þó byggð á svipuðum grunni.

10.apríl

Dælan var sótt, hún var síðan tæmd og þrifin. Það kemur alveg viðbjóðursleg lykt af sjálfskiptivökva þegar maður fær hann í föt. Síðan var byrjað að tengja dæluna og koma henni fyrir. Ég þurfti að sjóða bráðarbirgðarfestingu, smíða seinna úr áli svo ég geti soðið við innra brettin mín. Það boðaði ekki mikla lukku að kveikja á dælunni, mögulega ekki rétt tengt hjá mér.

11.apríl

Smá árgangur, dælan fór loksins í gang. Hún keyrir samt ekki á fullu afli. Kóðarnir sem ég fann á netinu virka ekki til að stilla hraðan á henni. Hún heldur sér bara á stöðum snúning.

12.apríl

Dæla tekin í sundur og séð hvort hægt sé að tengja framhjá örtölvunni. Það boðaði engan árangur, það virðast vera þrír transistorar og allir með mismunandi PWM merki.

13.apríl

Tíminn er að þrotum þannig ég keypti Velleman Arduino UNO starter pack með allskonar skynjurum og dóti. Ef ég næ engum árangri með CAN bus þá þarf ég að búa til eitthvað nýtt verkefni yfir helgina.

15.apríl

Þrjóskaðist eitthvað smá meira í CAN bus.

16.apríl

Brainstorming fyrir nýja hugmynd. Stal viftu úr borðtölvunni hans vinar míns. Leitaði um allt hús og bílskúr að einhverju sem tekur við PWM merki.

17/18.apríl

Fann laser skorinn kassa úr krossvið sem ég gerði fyrir tölvustudda framleiðslu. Viftan sem ég er með passar fullkomlega í þennan kassa. Ég tók meitil og barði út grófan úrskurð fyrir viftu og skjá. Tók síðan þjöl og fínslípaði og boraði göt fyrir skrúfur. Ég bjó til grófa mynd af rafkerfinu fyrir Arduino tölvuna og byrjaði að setja saman einingarnar. Þegar ég sá að allt virkaði tróð ég öllu í kassan. Fannst verkefnið allt of einfalt þannig ég bjó til spjald þannig hægt væri að opna og loka kassanum. Boraði síðan tvö göt sem ég gat komið plaststrappa í gegn og notaði hann til að festa Servó mótor beint á hliðina á kassanum. Síðan notaði ég vír úr Arduino settinu sem ég átti til að festa spjaldið við Servo motorinn og notaði ryðfrían TIG fyllivír sem ég á nóg af til að festa spjaldið við kassan. Nú getur spjaldið opnast og lokast. Eftir svona 20 kaffibolla og engan svefn náði ég að koma öllu saman, tók blund í smá stund og skellti mér síðan upp í skóla. Skoðaði frábæru verkefnin hjá samnemundunum mínum upp í VR2 og stóð vandræðalegur með kassann minn meðan gert var viðtal við þá. Kynningin gekk furðuvel þrátt fyrir að hafa verið mjög stutt. Held ég hafi verið of þreyttur til að vera stressaður :)

Servo myServo;  // create a servo object

LiquidCrystal lcd(12, 11, 5, 4, 3, 2);




const int fanPin = 9; // connected to pin 7 on the H-bridge

//Temp
// named constant for the pin the sensor is connected to
const int sensorPin = A0;
// room temperature in Celsius
const float baselineTemp = 20.0;

int angle;   // variable to hold the angle for the servo motor

void setup() {
  // put your setup code here, to run once:
  myServo.attach(10); // attaches the servo on pin 9 to the servo object
  // open a serial connection to display values
  pinMode(fanPin, OUTPUT);
  Serial.begin(9600);
  // set up the number of columns and rows on the LCD
  lcd.begin(16, 2);
    // Print a message to the LCD.
  lcd.print("XxXxX");
  // set the cursor to column 0, line 1
  // line 1 is the second row, since counting begins with 0
  lcd.setCursor(0, 1);
  // print to the second line
  lcd.print("xXxXx");

}

void loop() {
   // read the value on AnalogIn pin 0 and store it in a variable
  int sensorVal = analogRead(sensorPin);
  lcd.clear();

  // send the 10-bit sensor value out the serial port
  Serial.print("sensor Value: ");
  Serial.print(sensorVal);

  // convert the ADC reading to voltage
  float voltage = (sensorVal / 1024.0) * 5.0;

  // Send the voltage level out the Serial port
  Serial.print(", Volts: ");
  Serial.print(voltage);

  // convert the voltage to temperature in degrees C
  // the sensor changes 10 mV per degree
  // the datasheet says there's a 500 mV offset
  // ((voltage - 500 mV) times 100)
  lcd.setCursor(0, 0);
  lcd.print("Degrees C: ");
  float temperature = (voltage - .5) * 100;
  lcd.print(temperature);
  angle = 179-map(temperature, 15, 30, 0, 179);
  Serial.print(", angle: ");
  myServo.write(angle);
  lcd.setCursor(0, 1);
  lcd.print("Angle: ");
  lcd.print(angle);
  
  if(temperature>18) {
    digitalWrite(fanPin, HIGH);
    
  }
  else {
    digitalWrite(fanPin, LOW);
  }
  
  delay(1750);
}
